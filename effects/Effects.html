Effect development ​

Hyperion provides a powerful framework for developing your own effects, along with options and a user interface for tuning them.

Effect Files ​

An effect has 3 different files.

FilelanguageComment
neweffect.pyPythonThe heart of the effect
neweffect.jsonJSONContains options for the python file, which makes it configurable
